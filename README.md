# firstmile

Open-source, self-hosted, position-only instrumentation for watching onboarding flows in real time.

The complete kit, projector dashboard, sidecar, API reference, and quickstarts are in [`packages/kit`](packages/kit/README.md). A framework-free integration lives in [`examples/plain-html`](examples/plain-html/README.md).

The demo application is intentionally reserved for Prompt 2.
